##  專案環境說明

> 使用 React + Vite 建構，支援 RWD，部署於 GitHub Pages。

---

##  作品發想由來

這個作品的初衷，是希望將過往作品集中在同一個網頁上，方便瀏覽者一次性了解個人經歷與能力。整體發想以個人為商品，透過「商品介紹」的方式呈現，讓瀏覽者能更直觀地認識個人特質與專業背景。

由於培訓期間主要使用 Bootstrap，結束後自行學習 Tailwind，並嘗試從設計配色到前端開發皆由自己完成。資料內容建立在 JSON 檔案中，方便後續只需修改檔案即可更新內容。

---

##  使用技術

| 類別 | 技術 |
|------|------|
| 框架 | React 18 |
| 建構工具 | Vite |
| UI 元件 | shadcn/ui |
| 樣式 | Tailwind CSS、RWD |
| 程式碼品質 | ESLint + Prettier |
| 部署 | GitHub Pages |

---

## 專案結構

```
portfolio-noraDev/
├── public/               # 靜態資源
├── src/                  # 原始碼
│   ├── components/       # React 元件
│   ├── pages/            # 頁面
│   └── assets/           # 圖片、字體等資源
├── portfolioData.json    # 作品集資料設定檔
├── components.json       # shadcn/ui 元件設定
├── vite.config.js        # Vite 設定
└── index.html            # 進入點
```
